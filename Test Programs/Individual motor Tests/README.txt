Releaser(s):
Sauhaarda and Helen

Release Notes:
This suite of tests test each motor separately. MotorDTest, MoterGTest, MoterETest and MoterFTest will
spin each wheel forward, ArmTest will raise the arm up and then lower the arm down. These test need can
be run to indentify which moter is having problem in case of robot movement problems.
